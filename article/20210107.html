用Hexo的时间不是很长，但是用了一段时间真的非常喜欢。不过由于Github使用不是很熟练，所以经常过一段时间不用就忘了命令和一些常需要修改的参数，所以在这篇文章里面记录下来。

最常用

  • 生成网站

hexo g # 生成全站静态文件

hexo generate # 完整写法

生成或者更新 public 文件夹

  • 本地预览

hexo s # 启动本地预览服务

hexo server # 完整写法

启动服务器，访问网址： http://localhost:4000/ 进行本地预览。

  • 部署

hexo d # 部署网站到 Github 。

hexo deploy # 完整写法

把本地的网站推送到 Github ，完成推送后，需要重新到项目设置处绑定域名。

  • 清理缓存

hexo clean # 清除缓存文件 (db.json) 和已生成的静态文件 (public)

部分情况（比如更换主题后），发现站点的更改不生效，可以试试运行该命令。

  • 创建文章

hexo n "我的博客" # 新建文章

hexo new "我的博客" # 完整写法

新建一篇名字为我的博客的文章。我一般是用 Typora 写文章，所以这个命令使用的不多。

  • 检查版本

hexo version # 显示 hexo 版本
